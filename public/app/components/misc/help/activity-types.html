integer.divisors

Actividad de cálculo de los divisores de un número.
Parámetros
num Número del que debe hallarse los divisores
Método de entrada Lista con botones +, -
Ejemplo
{ "title": "Cálculo de divisores de un número", "parameters": { "num": "Math.irandom(2, 30)" }, "repeat": 10, "type": "integer.divisors" }

O un código equivalente, excepto que permite la configuración del parámetro maxValue:

"config": { "maxValue": 30 }, "questions": [ { "title": "Cálculo de divisores de un número", "parameters": { "num": "Math.irandom(2, config.maxValue)" }, "repeat": 10, "type": "integer.divisors" } ]

integer.gcd

Actividad de el máximo común divisor de una lista de números.
Parámetros
list Lista de números de los cúales debe hallarse el max.c.d
Método de entrada Campo de texto
Ejemplo
{ "title": "Cálculo del max.c.d", "parameters": { "x1": "Math.irandom(2, 30)", "x2": "Math.irandom(2, 30)", "list": "[this.x1, this.x2]" }, "repeat": 10, "type": "integer.gcd" }

integer.lcm

Actividad de el mínimo común múltiplo de una lista números.
Parámetros
list Lista de números de los cúales debe hallarse el min.c.m
Método de entrada Campo de texto
Ejemplo
{ "title": "Cálculo del min.c.m", "parameters": { "x1": "Math.irandom(2, 30)", "x2": "Math.irandom(2, 30)", "list": "[this.x1, this.x2]" }, "repeat": 10, "type": "integer.lcm" }

integer.primes

Actividad de seleccionar los números primos de una lista.
Parámetros
list Lista de números de los cuáles se deben seleccionar los primos
Método de entrada Botones de selección
Ejemplo
{ "title": "Números primos", "parameters": { "list": "list(10, rent1(0,50,false,false))" }, "repeat": 10, "type": "integer.primes" }

integer.divisibility

Actividad de seleccionar los números divisibles por uno dado de una lista.
Parámetros
list Lista de números de los cuáles se deben seleccionar cúales son divisibles
num Número por el cúal se debe comprobar la divisibilidad
Método de entrada Botones de selección
Ejemplo
{ "title": "Criterios de divisibilidad por 2, 3 y 5", "parameters": { "num": "[2, 3, 5].random()", "list": "list(10, rent1(0,50,false,false))" }, "repeat": 10, "type": "integer.divisibility" }

integer.operations

Esta tipo de actividad genera operaciones combinadas com números enteros.
Parámetros Uno de los dos parámetros:
template Plantilla de operación.
expr Expresión matemática
Método de entrada Campo de texto
Ejemplo
{ "title": "Realiza la siguiente operación", "parameters": { "template": "['N+Z0*(Z+Z-N)', 'Z0+Z0+Z0'].random()" }, "repeat": 10, "type": "integer.operations" }
o mediante el uso del parámetro expr
{ "title": "Realiza la siguiente operación", "parameters": { "expr": "RG1.rent1(2, 20, true, true)" }, "repeat": 10, "type": "integer.operations" }

equation

Esta tipo de actividad genera ecuaciones.
Parámetros Uno de los dos parámetros:
template Plantilla de la ecuación (procesada como un template).
range Rango para los valores aleatorios de la template
eq Expresión matemática de la ecuación
bar Nombre de la variable (por defecto x)
Método de entrada Lista de campos de texto con botones +/-
Ejemplo
{ "title": "Ecuaciones de segundo grado incompletas", "parameters": { "template": "['Z0*x^2+Z*x=0', 'Z0*x^2=Z0*Z_1*Z_1'].random()", "range": 10 }, "repeat": 10, "type": "equation" }
o mediante el uso del parámetro eq
{ "title": "Ecuaciones de segundo grado incompletas", "parameters": { "a": "Math.irandom(1,10)", "b": "Math.irandom(-10,10)", "eq": "this.a+'*z^2'+this.b+'*z=0'", "bar": "z" }, "repeat": 10, "type": "equation" }
Alternativamente, tambien se pueden utilizar las funciones aleatorias del paquete RGALG. Por ejemplo, RGALG.rpolz(g, cos, min, max) : genera un polinomio de grado g, con raices en cuerpo cos='Z' o 'Q', entre valores min y max. Entonces, se podria definir la ecuación "eq" como "eq": "RGALG.rpolz(2, 'Z', -10, 10).toString('x')+'=0'" para definir una ecuación de 2º grado com raices enteras.

polynomial.operations

Esta tipo de actividad genera operaciones con polinomios.
Parámetros Uno de los dos parámetros:
template Plantilla de la operación (procesada como un template).
range Rango para los valores aleatorios de la template
expr Expresión matemática de la operación
bar Nombre de la variable (por defecto x)
Método de entrada Mathquill
Ejemplo
{ "title": "Suma y resta de polinomios", "parameters": { "n1": "rent1(0,4,false,false)+1", "n2": "rent1(0,4,false,false)+1", "poly1": "rpolc(#n1,'Z',-20,20)", "poly2": "rpolc(#n2,'Z',-20,20)", "op": "['+','-'].random()", "expr": "#poly1.toString('x')+#op+'('+#poly2.toString('x')+')'" }, "repeat": 10, "steps": [ { "type": "polynomial.operations", "formulation": "'P(x)='+#poly1.toString('x', 'html')+', Q(x)='+#poly2.toString('x','html')+'. Calcula P(x) '+#op+' Q(x)'", "tips": "Recorda que pots sumar nombre amb nombre, x amb x, x^2 and x^2, etc." } ] }
