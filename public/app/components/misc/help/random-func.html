Funciones aleatorias

Para dar un carácter aleatorio a las actividades se utilizan una serie de funciones aleatorias.

[ ].random()

Si se tiene una lista de objetos y se quiere obtener un elemento aleatoriamente, se puede utilizar el método random() sobre el objeto lista. El siguiente ejemplo muestra como elegir aleatóriamente una expresión de una lista.
"parameters": { "func": "['2*x+1','1/x','sqrt(x)'].random()" }

La expresión func será una de las cadena de texto de la lista.

[ ].shuffle()

El método shuffle, desordena los elementos de la lista de forma aleatoria.

list(n, expr)

Genera una lista de n elemntos a partir de la expresión aleatoria expr. Este ejemplo genera una lista de 10 números aleatorios.
"parameters": { "lista": "list(10, Math.irandom(2, 30))" }

Número aleatório entre a y b

Las siguientes opciones para crear un número aleatorio entero son equivalentes
"parameters": { "x1": "Math.irandom(0, 10)", "x2": "Math.floor(Math.random()*10)", "x3": "RG1.rent1(0, 10, false, true)" }
Math.irandom(a, b): Crea un número aleatorio entre a y b. RG1.rent1(n, vmax, z, c): Crea un número aleatorio si n=0, de valor máxima vmax, pudiendo ser negativo si z=true, y incluyendo el cero si c=true. Para valores mayores de "n", genera operaciones aleatorias con números enteros con mayor grado de dificultat.

Plantillas o templates

RG1.processTemplate(expr, {range: value})
La expresión expr, puede contener los siguentes paramétros N: Número natural incluyendo el cero N0: Número natural sin el cero Z: Número entero incluyendo el cero Z0: Número entero sin el cero range, determina el valor máximo para los números aleatorios. Ejemplos:
"expr": "RG1.processTemplate('N0-2*Z+N0*(Z+Z-Z)', {range: 20})"
